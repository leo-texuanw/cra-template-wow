# cra-template-wow

A custom template for React App.

## What's included?

- Typescript
- Jest
- Eslint

## Testing template locally

```
npx create-react-app my-app --template file:../path/to/cra-template-wow
```

or

```
yarn create react-app my-app --template file:../path/to/cra-template-wow
```
